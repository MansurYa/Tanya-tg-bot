## 🔄 Логика утренних сообщений

1. После завершения квеста активируется `morning_messages_enabled=1`
2. Планировщик генерирует случайное время (09:45 ± 67 мин)
3. Бот отправляет комбинированное сообщение (emotional + psychological)
4. Трекинг показанных ID в БД (`emotional_pool_shown_ids`, `psychological_pool_shown_ids`)
5. Кнопка "Ещё одно сообщение" для дополнительных сообщений
6. Автоматический сброс цикла при исчерпании
