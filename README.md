# Tanya Telegram Bot

Персонализированный Telegram бот с системой квестов и утренними сообщениями.

## Возможности

- **Система квестов**: Интерактивный квест с 12 подарками и вопросами
- **Утренние сообщения**: Ежедневные автоматические сообщения с комплиментами и психологической поддержкой
- **Поддержка**: Эмоциональная поддержка и виртуальные объятия по запросу
- **Ротация сообщений**: Умная система ротации без повторов внутри цикла

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте бота:
```bash
cp config.example.json config.json
# Отредактируйте config.json, добавив токены и API ключи
```

3. Запустите миграцию базы данных:
```bash
python migrate_db.py
```

4. Запустите бота:
```bash
python -m src.bot
```

## Структура проекта

```
/src
  ├── bot.py              # Основной модуль бота
  ├── database.py         # Работа с SQLite
  └── message_pool.py     # Логика ротации сообщений

/tanya_dataset_generator
  ├── output_v2/          # Финальные датасеты
  ├── stages_v2/          # Скрипты генерации датасетов
  └── utils_v2/           # Утилиты для генерации

config.json              # Конфигурация бота (не в git)
data.db                  # База данных SQLite (не в git)
```

## Безопасность

- Никогда не коммитьте `config.json` с реальными токенами
- Используйте `config.example.json` как шаблон
- Храните `data.db` только локально
